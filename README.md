# Weather App

A weather application built as part of The Odin Project curriculum. Users can search for a location and view current weather information through a clean and responsive interface.

## Features

* Search weather by city name
* View current temperature and weather conditions
* Display additional information such as humidity, wind speed, and feels-like temperature
* Handle invalid searches with user-friendly error messages
* Responsive layout for desktop and mobile devices
* Built using JavaScript modules and Webpack

## Technologies Used

* HTML
* CSS
* JavaScript (ES6)
* Webpack
* Weather API (visualcrossing)

## What I Learned

This project helped me gain practical experience with:

* Fetching and displaying data from external APIs
* Working with asynchronous JavaScript using async/await
* Error handling in API requests
* Organizing code with ES6 modules
* Bundling projects with Webpack
* Deploying applications using GitHub Pages
